# 計画書フォーマットルール（Swift-Selena用）

## 目的

Swift-Selenaの開発計画を管理し、以下を実現します：

- **バージョン別計画**: v0.5.x系、v0.6.x系等の開発内容を明確化
- **マイルストーン管理**: リリース目標と進捗を可視化
- **要件との紐付け**: 各バージョンで実現する要件を明示

---

## Swift-Selenaの計画書

### PLAN.md

**目的:**
- v0.5.x〜v1.0の全体開発計画
- バージョン別実装内容
- マイルストーン管理

**構成:**
```markdown
# Swift-Selena 開発計画

**現在バージョン**: v0.X.X

## ロードマップ

### v0.X.X（ステータス）

**実装内容:**
1. [機能1]
2. [機能2]

**ツール総数:** XX個

**工数:** X時間

**リリース目標:** YYYY年MM月

---

## 機能別計画

### 1. [機能カテゴリ]

**段階:**
- v0.X.X: [実装内容]
- v0.Y.Y: [実装内容]

---

## マイルストーン

### YYYY年QX

- ✅ v0.X.X完了
- ⏳ v0.Y.Y進行中

---
```

## バージョン管理

**バージョン番号体系:**
- v0.5.x: LSP統合フェーズ
- v0.6.x: Code Header DBフェーズ
- v0.7.x: ベクトル検索フェーズ
- v0.8.x: 統計・分析フェーズ
- v1.0.0: 安定版リリース

**ステータス表示:**
- ✅ 完了
- ⏳ 進行中
- 📋 計画中
